HTML5,CSS3 Y JAVASCRIPT

HTML/CSS/JAVASCRIPT hoy en dia son los lenguajes de marcado y lenguajes de programacion enfocados al desarrollo web. Comenzando con HTML que sirve para darle la estructura o esqueleto a cualquier pagina o servicio web. CSS se considera el lenguaje de marcado para darle estilos, como los son colores, fuentes de texto, amigables vistass, alguno que otro efecto a elementos declarados en tu index.html de tu proyecto. Por ultimo tenemos a JS, el cual cumple con la funcion de darle la logica a una pagina o proyecto web, permite desarrollar funcionalidades que CSS no permite desarrollar. Control de versiones con Git

Hoy en día, Git es, con diferencia, el sistema de control de versiones moderno más utilizado del mundo. Git es un proyecto de código abierto maduro y con un mantenimiento activo que desarrolló originalmente Linus Torvalds, el famoso creador del kernel del sistema operativo Linux, en 2005. Un asombroso número de proyectos de software dependen de Git para el control de versiones, incluidos proyectos comerciales y de código abierto. En este apartado se encuentran algunos tips y formas de desarrollo, para un correcto control de versiones con Git

C/C++

VENTAJAS:

Compatibilidad con bibliotecas: A través de bibliotecas hay muchas funciones que están disponible y que ayudan a escribir código rápidamente.

Orientado a Objetos: El foco de la programación está en los objetos y la manipulación y configuración de sus distintos parámetros o propiedades.

Rapidez: La compilación y ejecución de un programa en C++ es mucho más rápida que en la mayoría de lenguajes de programación.

Compilación: En C++ es necesario compilar el código de bajo nivel antes de ejecutarse, algo que no ocurre en otros lenguajes.

Punteros: Los punteros del lenguaje C, también están disponibles en C++.
